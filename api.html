prfm.prfm.get_weights(Sigma_gas, Sigma_star, Omega_d, H_star, sigma_eff, zeta_d=0.3333333333333333, method='analytic')[source]#

calculate gas scale height and then each weight term

Parameters

  • Sigma_gas (float or array-like) – gas surface density

  • Sigma_star (float or array-like) – stellar surface density

  • Omega_d (float or array-like) – galactic rotation speed

  • H_star (float or array-like) – stellar scale height

  • sigma_eff (str or float or array-like) – velocity dispersion

  • zeta_d (float [1/3.]) – parameter for dm and gas distribution

  • method (str [analytic or numerical]) – calculate scale height either analytically or numerically

Returns

  • H_gas (float or array-like) – gas scale height

  • W_gas (float or array-like) – weight by gas

  • W_star (float or array-like) – weight by star

  • W_dm (float or array-like) – weight by dark matter

Example

>>> import prfm
>>> H, W_gas, W_star, W_dm = prfm.get_weights(Sigma_gas, Sigma_star, Omega_d, H_star, sigma_eff)
prfm.prfm.get_scale_height(Sigma_gas, Sigma_star, Omega_d, H_star, sigma_eff, zeta_d=0.3333333333333333, method='analytic', wgas=1, wstar=1, wdm=1)[source]#

wrapper function to calculate gas scale height either numerically or analytically

Parameters

  • Sigma_gas (float or array-like) – gas surface density

  • Sigma_star (float or array-like) – stellar surface density

  • Omega_d (float or array-like) – galactic rotation speed

  • H_star (float or array-like) – stellar scale height

  • sigma_eff (str or float or array-like) – velocity dispersion

  • zeta_d (float [1/3.]) – parameter for dm and gas distribution

  • method (str [analytic or numerical]) – calculate scale height either analytically or numerically

  • wgas (int [0 or 1]) – toggle weight term from gas

  • wstar (int [0 or 1]) – toggle weight term from stars

  • wdm (int [0 or 1]) – toggle weight term from dark matter

prfm.phangs.compute_prfm_inputs(table)[source]#

Add PRFM-relevant derived columns to a megatable.

Derived columns added#

Sigma_gas

Total gas surface density = Sigma_mol + Sigma_atom [M_sun / pc^2].

e_Sigma_gas

Uncertainty on Sigma_gas = sqrt(e_Sigma_mol^2 + e_Sigma_atom^2) [M_sun / pc^2]. Added only when both e_Sigma_mol and e_Sigma_atom are present.

Omega_d

Angular velocity = V_circ_CO21_URC / r_gal [km / s / kpc].

H_star

Stellar scale height = Sigma_star / (2 * rho_star_mp) [pc].

param table

type table

~astropy.table.Table

returns

A copy of the input table with the new columns appended.

rtype

prfm.phangs.run_prfm(table: Table, prfm_cols: Optional[str] = ['Sigma_gas', 'Sigma_star', 'Omega_d', 'H_star'], sigma_eff_model: str = 'tigress-ncr-avg', yield_model: str = 'tigress-ncr-decomp-all', zprime_col: Optional[str] = 'Zprime', variation: Optional[dict] = None) → Table[source]#

Apply the PRFM model to every row of a PHANGS megatable.

Runs compute_prfm_inputs() if Sigma_gas is not already present, then calls the self-consistent PRFM solver (vectorized over all rows). Invalid rows (NaN or non-positive PRFM inputs) receive NaN outputs.

Columns added#

P_weight

Dynamical equilibrium pressure (total gas weight) in units of \(k_\mathrm{B}\) K cm-3.

H_gas

Self-consistent gas scale height [pc].

sigma_eff_sol

Self-consistent effective velocity dispersion [km s-1].

Sigma_SFR_pred

Predicted SFR surface density [M_sun kpc-2 yr-1].

param table

PHANGS megatable, optionally pre-processed by compute_prfm_inputs().

type table

~astropy.table.Table

param sigma_eff_model

Velocity dispersion model name (key in prfm._sigma_eff_models).

type sigma_eff_model

str

param yield_model

Feedback yield model name (key in prfm._yield_models).

type yield_model

str

param zprime_col

Column name for metallicity (relative to solar). Pass None to ignore metallicity dependence.

type zprime_col

str or None

returns

Copy of the input table with four new columns appended.
